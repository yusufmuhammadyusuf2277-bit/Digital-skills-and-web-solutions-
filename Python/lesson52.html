Lesson 52: Automated Network Health Checker

This script checks if devices on your network respond and logs results.

import os
import datetime

devices = ["192.168.1.1", "192.168.1.2"]
for ip in devices:
  response = os.system(f"ping -c 1 {ip}")
  status = "Online" if response == 0 else "Offline"
  print(f"{datetime.datetime.now()} - {ip} is {status}") Next Lesson →
