Lesson 57: Automated Log Monitoring

Automatically check logs for suspicious activity.

log_file = "system.log"
with open(log_file) as f:
  for line in f:
    if "failed" in line:
      print(f"⚠️ Suspicious activity: {line.strip()}")

Automated monitoring helps detect issues quickly.
